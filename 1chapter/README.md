## HTTP

- 먼저 인터넷 부터 이해, 네트워크의 집합체임
- 서로 통신하기 위해 HTTP를 쓴다
- HTTP는 어떤 종류의 데이터도 전송 가능하다
- 유상태? 지속적 / 채팅, 온라인 겜
- 무상태 필요할 때만 씀 / 메일?
- HTTP == 무상태
- 정보 보존을 위해 쿠키를 씀
- URL 위치> 프로토콜의 종류, IP, 도메인 주소, 포트번호, 문서의 경로
- 웹 동작, 클라이언트 요청
- ? 뒷부부분 무엇
